Solar Scorecard is a project of Code for Fort Collins. We're a small group of volunteers who build cool things that we think will help our city. In this specific case, we thought that it would be good to help the city monitor its progress on the Climate Action Goals.

Get Involved

This is a Code for Fort Collins project. You can find all its code on GitHub, and pull requests are more than welcome. If that's too complicated for you, find them on Meetup and attend an event to get involved in making it better :)

Contact

This project is maintained by Ian Skor from Sandbox Solar and the members of Code for Fort Collins. Please feel free to reach out to either (or both) of us about anything related to this project.
